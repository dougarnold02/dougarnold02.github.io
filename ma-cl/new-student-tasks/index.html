Some Tasks for new students on the MA in CL

Doug Arnold
University of Essex
doug(at)essex.ac.uk

October 2, 2007

The idea of these tasks is to get you familiar with a few things you will need to be familiar with (e.g. the computing labs, the library, etc. etc). They are not to be taken too seriously, but doing them will be useful.

  1. find all the computer labs mentioned on the timetable, in each one log on to a computer, start up a web browser, and find this page (You can get to it from my home page, which you will have to find for yourself).
  2. Send me (doug@essex.ac.uk) an email containing the text in this page (don't re-type it; don't bother too much about the format)
  3. Do a web search for the following:
    • Computational Linguistics
    • LFG
    • Constraint Based Grammar
    • Parsing
    • Prolog Natural Language
  4. Locate the following paper on the Web: Dalrymple, Mary. 2006. 'Lexical Functional Grammar'. In Keith Brown (eds), Encyclopedia of language and linguistics (2nd edition). Oxford: Elsevier. (Of course, it can also be found in the library).
  5. Find the name of a computational linguistics conference.
  6. Print this page.
  7. What is the most recent edition of the Computational Linguistics in the library? What is the first?
  8. What floor(s) are the books about Prolog on?
